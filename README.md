# 🌦️ Weather Map Web App

A modern dark-mode weather web application that shows real-time weather data for any city using the OpenWeather API.

## 🚀 Features

- 🔎 Search weather by city name  
- 🌡️ Real-time temperature display  
- 💧 Humidity and 🌬 wind speed  
- 🌤️ Dynamic weather icons  
- 🌙 Dark glassmorphism UI  
- ⌨️ Enter key search support  
- ⚠️ Invalid city error handling  


## 🛠️ Built With

- HTML5  
- CSS3 (Glassmorphism UI)  
- JavaScript (ES6)  
- OpenWeather API

## ⚙️ How It Works

1. User enters a city name  
2. App calls OpenWeather API  
3. Weather data returned in JSON  
4. UI updates dynamically
